What is a security key?

a Security key, also known as an API key is a unique code that serves as an authentication mechanism for accessing certain features or services on a website. When you insert your security key in this website, you let this website get your information from mylms.vossie.net so that we can display it here.

Security note: I don't store these security keys anywhere but on your browser, and it's stored on the browser in the form of a cookie, so that you don't have to log in again every time you exit the site. Meaning I don't have access to your security key.
